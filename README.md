# TerrainBrushes
New Terrain Tools and Brushes for the Unity Terrain system.

## Included Tools
- Create Neighbor Terrain
- Raise/Lower Terrain Height
- Paint Texture
- Set Height
- Smooth Height
- Clone Stamp
- Mesh Stamp
- Bridge
- Erosion
  - Erode Height
  - Terrace Erosion
- Twist
- Sharpen Height
- Slope Flatten Height
- Smudge Height
